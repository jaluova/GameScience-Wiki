Challenge 1 : Try EasyX ​

0xFF EasyX是什么?能吃吗 ​

EasyX是一个针对C/C++的简单,易上手的图形库.

学习完它你将可以做出一些画面精美的能接收鼠标信息的小游戏.

(是的,如果你用的是windows系统的控制台,是无法接受鼠标信息的😭)

如果你已经学过了C语言数组、指针、函数、结构体等知识,EasyX的学习对你来说将如同其名一样"Easy".

安装EasyX的教程:EasyX官网

官网描述还是挺详细的,这里便不过多赘述

(推荐使用Visual Studio 2022)

0x01 简单上手 ​

接下来是一段演示代码

c++
#include<graphics.h>

int main() {
	initgraph(1280, 720);
	
	int x = 300;
	int y = 300;
    
	BeginBatchDraw();

	while (true) {
		ExMessage msg;
        
		while (peekmessage(&msg)) {
			if (msg.message == WM_MOUSEMOVE) {
				x = msg.x;
				y = msg.y;
			}
			cleardevice();
			solidcircle(x, y, 100);
		}
		FlushBatchDraw();
	}
	EndBatchDraw();

	return 0;
}

我们不必急于去理解每段代码的含义,可以先简单上手玩一玩😏
